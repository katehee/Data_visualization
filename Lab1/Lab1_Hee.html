Q1. Line graph and reading the graph¶

Please use the "bitcoin.csv" file to see the data. In this dataset, you can see Bitcoin prices from the start of 2014 to the start of 2022.

Look at the Bitcoin prices for each year. Which year began with the highest? Visualize the bitcoin prices over the years.

Tip: There is more than one right answer. Please use your imagination to visualize the most impactful plot.

In [ ]:
import pandas as pd 
import matplotlib.pyplot as plt
from matplotlib.pyplot import figure
from matplotlib.dates import DateFormatter

file = pd.read_csv("bitcoin.csv")
df = pd.DataFrame(file)
# print(df.columns)
df = df[["Date", "24h Open (USD)"]]
x = df["Date"].apply(pd.to_datetime)
y = df["24h Open (USD)"]

# Create figure and plot space
fig, ax = plt.subplots(figsize=(12, 12))

# Add x-axis and y-axis
ax.plot(x,y)

# Set title and labels for axes
ax.set(xlabel="Year",
       ylabel="24h Open (USD)",
       title="bitcoin prices over 2014-2022 period")

# Display only the Year from date 
date_form = DateFormatter("%Y") 
ax.xaxis.set_major_formatter(date_form)
plt.show()

#As shown on the graph, 2022 begins with the highest price.

Q2. Interpreting histograms¶

Here is a dataset of salaries ("salary.csv") for various jobs in Australia.

Each row of the dataset represents the average salary for that job (for each gender), so the counts are counts of jobs. Visualize the salaries with using histogram and answer following questions;

  1. Is the histogram bimodal or unimodal?
  2. Is the histogram right-skewed or left skewed?
  3. What is the most popular (frequent) salary bracket?
  4. Investigate whether there is an equal pay for men and women

Change the bin-size and observe how the histogram changes.

Which bin size selection seems best for you?

Answers to Q2

  1. The histogram is unimodal because it has one peak.
  2. The histogram is right-skewed.
  3. The most popular salary bracket is 25K to 50K.
  4. From the 'salary distribution for each gender' histogram chart, we see that Male histogram is slightly less right-skewed than female histogram. This tells us that men get slightly higher salary than women. Thus we can say that there is no equal pay.
